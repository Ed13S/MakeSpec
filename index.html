Project Input

▸ Gemini API key (required, stored only in your browser)

This site calls the Gemini API directly from your browser — nothing is sent to any server but Google's. Get a key at aistudio.google.com. Your key is saved in this browser's local storage only, so you won't need to re-enter it next time on this device.

▸ OpenRouter API key (optional fallback if Gemini fails or is rate-limited)

If Gemini errors out (rate limit, overload, etc.) and a key is set here, the site automatically retries the same request through OpenRouter instead of just failing. Also stored only in your browser. Leave the model field blank to use OpenRouter's auto-router, which picks a capable model for you.
